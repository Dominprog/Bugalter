FAQ по системе
Ответы на вопросы о функционале системы

Как ввести данные по отработанным часам сотрудника?

Для ввода данных по отработанным часам:

  1. Перейдите в раздел "Сотрудники"
  2. Выберите нужного сотрудника из списка
  3. Нажмите кнопку "Редактировать"
  4. В разделе "Отработанные часы" введите количество часов за каждый день месяца
  5. Нажмите "Сохранить" для применения изменений

Как посмотреть сотрудников со сверхурочными часами?

Чтобы найти сотрудников со сверхурочными часами:

  1. Перейдите в раздел "Сверхурочные"
  2. Система автоматически отобразит всех сотрудников с часами сверх нормы
  3. Вы можете отсортировать список по количеству сверхурочных часов или отделу
  4. Для детальной информации нажмите на имя сотрудника

Как рассчитывается зарплата за вычетом подоходного налога?

Расчет зарплаты с учетом налога:

  1. Система автоматически рассчитывает общую сумму заработка
  2. Из этой суммы вычитается 12% подоходного налога
  3. Для просмотра зарплаты с учетом налога перейдите в раздел "Отчеты"
  4. Выберите отчет "Зарплата за вычетом налога"
  5. Система отобразит список всех сотрудников с суммой к выплате

Как найти сотрудников с зарплатой ниже указанной?

Для поиска сотрудников с зарплатой ниже определенного уровня:

  1. Перейдите в раздел "Отчеты"
  2. Выберите опцию "Сотрудники с низкой зарплатой"
  3. Введите пороговое значение зарплаты
  4. Нажмите кнопку "Поиск"
  5. Система отобразит всех сотрудников с зарплатой ниже указанной

Как учитываются премии и единоразовые выплаты?

Учет премий и единоразовых выплат:

  1. Премии добавляются в разделе "Сотрудники" → "Дополнительные выплаты"
  2. Выберите тип выплаты: премия или единоразовая выплата
  3. Укажите сумму и дату выплаты
  4. Система автоматически учтет эти суммы при расчете общей зарплаты
  5. Премии и единоразовые выплаты также учитываются при расчете налога

Инструкции по работе с системой

Пошаговые руководства по основным функциям

Добавление нового сотрудника

Инструкция по внесению данных о новом сотруднике в систему

  • 1. Перейдите в раздел "Сотрудники"
  • 2. Нажмите кнопку "Добавить сотрудника"
  • 3. Заполните все обязательные поля
  • 4. Укажите тип оплаты (оклад/почасовая)
  • 5. Сохраните данные

Учет рабочего времени

Как правильно вводить данные об отработанных часах

  • 1. Откройте карточку сотрудника
  • 2. Перейдите во вкладку "Рабочее время"
  • 3. Введите часы по дням месяца
  • 4. Сверхурочные часы отмечаются автоматически
  • 5. Сохраните изменения

Расчет заработной платы

Процесс расчета зарплаты с учетом всех факторов

  • 1. Система автоматически рассчитывает зарплату
  • 2. Учитываются отработанные часы и ставка
  • 3. Добавляются премии и надбавки
  • 4. Вычитается подоходный налог (12%)
  • 5. Результат можно посмотреть в отчетах

Формирование отчетов

Создание различных отчетов по зарплате и рабочему времени

  • 1. Перейдите в раздел "Отчеты"
  • 2. Выберите тип отчета
  • 3. Укажите период формирования
  • 4. Настройте параметры отчета
  • 5. Сгенерируйте и сохраните отчет

Настройка ставок и тарифов

Изменение почасовых ставок и тарифов для сотрудников

  • 1. Откройте раздел "Настройки"
  • 2. Перейдите во вкладку "Тарифы"
  • 3. Выберите разряд сотрудника
  • 4. Установите новую ставку
  • 5. Сохраните изменения

Анализ данных

Использование аналитических инструментов системы

  • 1. Откройте раздел "Аналитика"
  • 2. Выберите интересующий показатель
  • 3. Настройте параметры анализа
  • 4. Просмотрите графики и диаграммы
  • 5. Экспортируйте данные при необходимости
